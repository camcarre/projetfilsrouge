    - Exporter `getRiskMetrics(ticker: string, period?: string): Promise<{ data: RiskMetrics | null; error: Error | null }>`
      → GET `/api/analyze/risk/${ticker}?period=${period ?? '1y'}`
    - Même gestion d'erreur que les autres services (try/catch, retourner { data: null, error })
    - Ne pas dupliquer la logique base URL — utiliser le client existant.
