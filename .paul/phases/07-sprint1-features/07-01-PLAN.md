<objective>
## Goal
Ajouter un bouton "Exporter CSV" dans la page Portfolio qui génère et télécharge un fichier CSV des actifs côté client sans appel serveur.

## Purpose
Fonctionnalité demandée dans le cahier des charges (section 3.2 Objectifs Secondaires : "Export des données et rapports"). Simple à implémenter, haute valeur pour la démo.

## Output
- `src/pages/Portfolio/PortfolioPage.tsx` modifié : bouton Export CSV + fonction generateCsv
</objective>

<acceptance_criteria>

## AC-1: Bouton Export visible
```gherkin
Given un utilisateur connecté avec des actifs dans son portfolio
When il visite la page Portfolio
Then un bouton "Exporter CSV" est visible dans le header de la page
```

## AC-2: Téléchargement déclenché
```gherkin
Given des actifs chargés dans le portfolio
When l'utilisateur clique sur "Exporter CSV"
Then un fichier portfolio_AAAA-MM-JJ.csv est téléchargé automatiquement
```

## AC-3: Contenu CSV correct
```gherkin
Given un portfolio avec plusieurs actifs
When le CSV est ouvert dans Excel/Sheets
Then il contient les colonnes : Nom, Symbole, Catégorie, Quantité, Prix achat, Prix actuel, Valeur totale, +/- %
```

## AC-4: Pas de téléchargement si portfolio vide
```gherkin
Given un portfolio sans actifs
When l'utilisateur clique sur "Exporter CSV"
Then rien ne se passe (bouton désactivé ou ignoré silencieusement)
```

</acceptance_criteria>

<task type="auto">
  <name>Ajouter la fonction generateCsv et le bouton Export dans PortfolioPage</name>
  <files>src/pages/Portfolio/PortfolioPage.tsx</files>
  <action>
    1. Ajouter une fonction `generateCsv(assets: Asset[])` dans le composant :
       - Header CSV : `Nom,Symbole,Catégorie,Quantité,Prix achat (€),Prix actuel (€),Valeur totale (€),+/- %`
       - Pour chaque actif : calculer `currentValue = asset.quantity * asset.unitPrice`, `pnlPct = ((unitPrice - originalPrice) / originalPrice * 100).toFixed(2)`
       - Encoder en `data:text/csv;charset=utf-8,` + encodeURIComponent
       - Créer un `<a>` temporaire avec `download=portfolio_${new Date().toISOString().slice(0,10)}.csv`
       - Cliquer puis retirer du DOM

    2. Ajouter le bouton dans le header de la page, à côté du bouton "Ajouter un actif" :
       - `<Button variant="secondary" onClick={() => generateCsv(assets)} disabled={assets.length === 0}>`
       - Texte : "Exporter CSV"
       - Visible uniquement si `!requiresAuth`

    Avoid : appels réseau, dépendances externes (papaparse etc.), modifier la structure du composant au-delà du bouton et de la fonction.
  </action>
