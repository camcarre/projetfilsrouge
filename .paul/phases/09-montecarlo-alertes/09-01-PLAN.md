<objective>
## Goal
Endpoint `GET /api/portfolio/montecarlo` qui simule N trajectoires de valeur du portefeuille de l'utilisateur sur un horizon T jours et retourne les percentiles (p5, p25, p50, p75, p95) + VaR Monte Carlo.

## Purpose
Donner à l'utilisateur une projection probabiliste de la valeur future de son portefeuille (cône de percentiles) pour visualiser le risque. Backend seul — le graphique frontend viendra en phase suivante.

## Output
- `backend/services/montecarlo_service.py` : simulation GBM à partir des rendements historiques
- Endpoint FastAPI authentifié
- Test pytest de non-régression
</objective>

  <name>Service Monte Carlo</name>

  <action>
Créer `simulate_portfolio(holdings, days, simulations)` où `holdings = [{"symbol", "quantity", "unit_price"}]`.
Algorithme (GBM par portefeuille agrégé, numpy vectorisé) :
1. Récupérer l'historique 1y de chaque symbole via `yf.download` ou `yf.Ticker(s).history(period="1y")` (Close).
2. Aligner les dates (intersection), construire la série de valeur du portefeuille = Σ quantity_i × close_i,t.
3. Rendements log journaliers → mu (moyenne) et sigma (écart-type).
4. Valeur initiale V0 = Σ quantity_i × dernier close_i (fallback unit_price live si close manquant).
5. Simuler `simulations` trajectoires sur `days` : matrice numpy (simulations × days) de chocs normaux N(mu, sigma), cumul produit → V0 × exp(cumsum). Vectorisé, PAS de boucle Python par trajectoire.
6. À chaque pas t, calculer np.percentile [5,25,50,75,95] sur l'axe des simulations.
7. `var95` = V0 − percentile_p5 du dernier pas (perte potentielle à 95% de confiance sur l'horizon).
Retour dict : `{ "initialValue", "days", "simulations", "steps": [{"day", "p5","p25","p50","p75","p95"}], "var95", "var95Pct" }`.
Gestion erreurs : lever ValueError si aucun historique exploitable (≥2 points communs). Type hints obligatoires (rule python.md). Pas de print → laisser remonter l'exception, l'endpoint la traduit.
  </action>
  <verify>python -c "from backend.services.montecarlo_service import simulate_portfolio; r=simulate_portfolio([{'symbol':'AAPL','quantity':10,'unit_price':150}],30,500); assert r['steps'][-1]['p5']<=r['steps'][-1]['p95']; print('OK', r['var95'])"</verify>
  <done>AC-1, AC-4 satisfaits : simulation vectorisée retourne percentiles ordonnés + var95.</done>
