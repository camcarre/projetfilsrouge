    **Fonctions à implémenter :**

    `validate_ticker(ticker: str) -> bool`
    - Retourner `bool(re.match(r'^[A-Z0-9]{1,10}(\.[A-Z]{2})?$', ticker or ''))`

    `_calc_match(ter: float, esg: str) -> int`
    - Port exact de `calcMatch()` JS : terScore = max(0, 100 - ter*50) ; esgOrder = ['B','A','AA','AAA'] ; esgScore = 60 + idx*13 si trouvé sinon 70 ; retourner round(terScore*0.4 + esgScore*0.6)

    `_calculate_volatility(historical: list[dict]) -> float`
    - Port exact de `calculateVolatility()` JS : rendements journaliers, variance, sqrt(variance * 252) * 100

    `get_etf_details(ticker: str) -> dict`
    - `ticker_obj = yf.Ticker(ticker)`
    - quote via `ticker_obj.fast_info` (last_price, previous_close, etc.) — try/except
    - historical via `ticker_obj.history(period="1y", interval="1d")` — convertir en liste de dicts {date, open, high, low, close, volume}
    - sustainability via `ticker_obj.get_sustainability()` — retourner None si échoue
    - Retourner `{quote: {regularMarketPrice: ..., previousClose: ..., longName: ..., ...}, historical: [...], esgScore: None, sustainability: None}`

    `get_etf_performance(ticker: str, period: str = "1y") -> Optional[dict]`
    - Mapper period → yfinance period string : '1y'→'1y', '6m'→'6mo', '3m'→'3mo', '1m'→'1mo'
    - `hist = yf.Ticker(ticker).history(period=mapped_period)`
    - Si < 2 points : retourner None
    - perf = (last - first) / first * 100
    - volatility = `_calculate_volatility(hist_as_dicts)`
    - Retourner `{performance: round(perf, 2), volatility: round(vol, 2), data: [...]}`

    `get_etf_holdings(ticker: str) -> Optional[dict]`
    - `yf.Ticker(ticker).get_institutional_holders()` — retourner en dict ou None si échoue
    - try/except autour de tout

    `get_etfs(filters: dict = {}) -> list`
    - Si TICKERS vide : retourner []
    - Récupérer quotes individuellement : `[yf.Ticker(t).fast_info for t in TICKERS]` avec try/except par ticker
    - Si aucun résultat : fallback statique (même logique que JS — random price, isMock=True)
    - Construire la liste de dicts : {id, name, ticker, lastPrice, volume, change, changePercent, ter, perf1y, esg, match, zone, theme}
      - `ter = info.get('netExpenseRatio', 0.2) * 100` si dispo sinon 0.2
      - `perf1y = info.get('fiftyTwoWeekHigh', 0)` → utiliser `fast_info.fifty_two_week_high` si dispo
      - `change = last_price - prev_close` si prev_close dispo sinon 0
    - Filtrer par `zone`, `sector` (= theme), `esg` (ordre ['B','A','AA','AAA'] ≥ filtre), `terMax`
    - Retourner la liste filtrée

    Note : `fast_info` expose : `last_price`, `previous_close`, `last_volume`, `fifty_two_week_high`, `fifty_two_week_low`.
    Pour `longName` / `shortName` : utiliser `yf.Ticker(ticker).info.get('longName', ticker)` — mais plus lent.
    Compromis : utiliser META[ticker]['description'] comme name en priorité, sinon ticker.

    **Cache helper :**
    ```python
    def _get_cached(key: str, fn, *args, **kwargs):
        entry = _cache.get(key)
        if entry and (time.time() - entry['ts']) < CACHE_DURATION:
            return entry['data']
        data = fn(*args, **kwargs)
        _cache[key] = {'data': data, 'ts': time.time()}
        return data
    ```

      n = len(prices)
      x = np.arange(n)
      slope, intercept = np.polyfit(x, prices, 1)
      residuals = prices - (slope * x + intercept)
      std_dev = np.std(residuals)
      forecast = [round(float(slope * (n + i) + intercept + np.random.uniform(-1, 1) * std_dev * 0.75), 2) for i in range(30)]
      model_used = "linear-regression"
