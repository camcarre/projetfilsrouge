<task type="auto">
  <name>Task 2 : main.py — app FastAPI + CORS + logs + routes auth</name>
  <files>backend/main.py</files>
  <action>
    Créer `backend/main.py` qui réplique les parties auth de `backend/server.js` :

    **Setup app :**
    ```python
    from fastapi import FastAPI, Header, HTTPException, Depends
    from fastapi.middleware.cors import CORSMiddleware
    import uvicorn, bcrypt, os, secrets
    from dotenv import load_dotenv
    from database import get_db
    from pydantic import BaseModel

    load_dotenv()  # charge .env à la racine
    app = FastAPI()
    app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_methods=["*"], allow_headers=["*"])
    ```

    **Middleware log** (équivalent du `app.use` de log dans server.js) :
    - Utiliser un middleware Starlette ou un simple `@app.middleware("http")` qui log `method path`

    **Token helper** :
    - `random_token()` → `secrets.token_hex(32)` (plus secure que le JS Math.random)
    - `get_user_id(authorization: str = Header(None))` → extrait Bearer token, query sessions, retourne user_id ou None

    **Routes auth** (4 routes — reproduire exactement la logique de server.js) :
    - `POST /auth/register` : valide email+password, check doublon email, hash bcrypt(10), insert user + session, return {user, token}
    - `POST /auth/login` : récupère user, bcrypt compare, crée session, return {user, token}
    - `GET /auth/me` : Bearer token → user depuis DB, return {user: {id, email, displayName, knowledgeLevel}}
    - `POST /auth/logout` : supprime session, return 204

    **Démarrage** :
    ```python
    if __name__ == "__main__":
        uvicorn.run("main:app", host="0.0.0.0", port=3000, reload=True)
    ```

    Éviter : FastAPI `Depends` complexe pour l'auth — un simple Header() helper suffit pour rester proche du JS original.
